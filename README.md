# GlucoseGuide: Insulin Dose Calculator
Com Apps Final Report

## Overview
This program prompts the user to input their current blood glucose, target blood glucose, carbohydrates in meal, insulin to carb ratio, insulin sensitivity factor, activity level, and insulin pen increment. Using these inputs, the program generates a standard and conservative dose for the user to compare. It also highlights the difference between the two dosing strategies. Finally, the user has the option to view their insulin doses and blood glucose levels over time.

## Files
- Comp_apps_project.ipynb

## How to Run
This code is written in Python and uses matplotlib. 
No additional files are required to be ran before or after.

1. Open notedbook in Google Colab.
2. Click run.
3. Follow the prompts and input values.

## Output
- Standard insulin dose
- Conservative insulin dose
- Difference between 2 dosing strategies
- Safety warning if blood glucose is too low
- Session summary and history
- Visualization of standard dose, conservative dose, and blood glucose levels over time
